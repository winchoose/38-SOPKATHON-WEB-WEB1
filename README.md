<h2> 모시모시 </h2>

<p>말로 전하기 어려운 마음을 모스부호로 전달하는 웹 서비스</p>

<h2> ✨ 모시모시 주요 기능 </h2>

<h3> 1️⃣ 텍스트 입력 및 모스 변환 </h3>
<div >사용자가 전달하고 싶은 감정을 자유롭게 작성하면, 입력된 문장을 실시간으로 모스부호로 변환합니다. 단순한 텍스트 전달이 아닌 ‘해독 과정’을 통해 감정을 더욱 특별하게 전달할 수 있습니다.<br/></div>

<h3> 2️⃣ 고유 링크 생성 및 공유 </h3>
<div >변환된 모스부호는 고유 URL 형태로 생성되며, 카카오톡·DM·문자 등을 통해 간편하게 공유할 수 있습니다. <br/></div>

<h3> 3️⃣ 모스 편지 아카이빙 </h3>
<div >전달된 모스부호 편지는 서버에 영구 저장되며, 디지털 공간 속 감정 기록 보관함 역할을 합니다.<br/></div>

<h3> 4️⃣ 모스 답장 기능 </h3>
<div >수신자는 받은 편지에 대해 다시 모스부호로 답장할 수 있으며, 감정을 주고받는 양방향 소통 경험을 제공합니다.<br/></div>

<h2>  📄 컨벤션 및 브랜치 전략 </h2>

<h3> 컨벤션 </h3>

| 커밋 컨벤션 | 용도 |
| ----------- | ---- |
| feat        | 새로운 기능 추가 |
| refactor    | 코드 리팩토링 |
| fix         | 버그 수정 |
| style       | 코드 formatting, 세미콜론 누락, 코드 자체의 변경이 없는 경우 |
| name        | 파일 혹은 폴더명을 수정하는 작업 |
| file        | 파일을 삭제하는 작업만 수행한 경우 |
| remove      | 파일을 삭제하는 작업만 수행한 경우 |
| comment     | 필요한 주석 추가 및 변경 |
| docs        | 문서 수정 |
| chore       | 패키지 매니저 수정, 그 외 기타 수정 ex) .gitignore |

<h3> 브랜치 </h3>

| 브랜치 | 용도 |
| ------ | ---- |
| main | 최종 배포 |
| develop | 개발 통합 |
| feat/~ | 기능 개발 |
| fix/~ | 버그 수정 |
